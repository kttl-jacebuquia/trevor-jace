CookieHub is a complete Cookie Consent Solution that you can easily integrate on your website. The widget can be customized to match your brand and is compliant with cookie laws, including GDPR.

== Description ==

CookieHub is fully featured cookie consent management platform (CMP) that you can easily implement to your web site, giving your end users the choice to allow or disallow cookies. The cookie scanner will detect which cookies are in use on the web site, categorize them and automatically generate a cookie declaration listing all cookies in use and the purpose of each cookie.

Major features in CookieHub include:

* Automatic cookie scanning
* Automatic cookie declaration
* Customizable user interface
* Google Tag Manager integration
* Geolocation
* Consent Log
* Multiple Languages

== Installation ==

* Register for an account on dash.cookiehub.com and create your domain.
* Add the CookieHub Wordpress plugin to your site.
* Activate the plugin.
* Enter your CookieHub domain code.
